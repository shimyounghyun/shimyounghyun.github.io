- 테스트 클래스 생성및 이동
    - 커맨드 + shift + T

- Import 정리
    - 옵션 + 컨트롤 + o

- 좌측 네비게이션 이동
    - 커맨드 + 숫자

- 테스트 실행
    - 컨트롤 + 쉬프트 + R
- 실행 했던 테스트 실행
    - 컨트롤 + R
- 함수 추출
    - 커맨드 + 옵션 + m
- 변수 추출
    - 커맨드 + 옵션 + v
